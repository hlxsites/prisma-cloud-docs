=== Fix - Buildtime

*Terraform*

*Resource:* 
- azurerm_cdn_endpoint

*Arguments:* 
- is_https_allowed

[source,terraform]
----
resource "azurerm_cdn_endpoint" "example" {
  name                = "example-cdn-endpoint"
  # ... other configurations ...

+ is_https_allowed = true

  # ... other configurations ...
}
----
